<h2><a href="https://leetcode.com/problems/running-sum-of-1d-array/">1480. Running Sum of 1d Array</a></h2><h3>Easy</h3><hr>
<p>Given an array <code>nums</code>, return the running sum where <code>runningSum[i] = sum(nums[0]...nums[i])</code>.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> nums = [1,1,1,1]
<strong>Output:</strong> [1,2,3,4]</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> nums = [1,2,3,4]
<strong>Output:</strong> [1,3,6,10]</pre>
