<h2><a href="https://leetcode.com/problems/maximum-average-subarray-i/">643. Maximum Average Subarray I</a></h2><h3>Easy</h3><hr>
<p>Find a contiguous subarray of length <code>k</code> that has the maximum average value and return this value.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> nums = [1,12,-5,-6,50,3], k = 4
<strong>Output:</strong> 12.75000</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> nums = [5], k = 1
<strong>Output:</strong> 5.00000</pre>
