<h2><a href="https://leetcode.com/problems/reverse-vowels-of-a-string/">345. Reverse Vowels of a String</a></h2><h3>Easy</h3><hr>
<p>Given a string <code>s</code>, reverse only all the vowels in the string and return it.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> s = "IceCreAm"
<strong>Output:</strong> "AceCreIm"</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> s = "leetcode"
<strong>Output:</strong> "leotcede"</pre>
