<h2><a href="https://leetcode.com/problems/plus-one/">66. Plus One</a></h2><h3>Easy</h3><hr>
<p>You are given a large integer represented as an integer array <code>digits</code>. Increment the large integer by one and return the resulting array.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> digits = [1,2,3]
<strong>Output:</strong> [1,2,4]</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> digits = [9]
<strong>Output:</strong> [1,0]</pre>
