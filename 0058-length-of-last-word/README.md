<h2><a href="https://leetcode.com/problems/length-of-last-word/">58. Length of Last Word</a></h2><h3>Easy</h3><hr>
<p>Given a string <code>s</code> consisting of words and spaces, return the length of the last word in the string.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> s = "Hello World"
<strong>Output:</strong> 5</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> s = "   fly me   to   the moon  "
<strong>Output:</strong> 4</pre>
