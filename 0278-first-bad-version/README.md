<h2><a href="https://leetcode.com/problems/first-bad-version/">278. First Bad Version</a></h2><h3>Easy</h3><hr>
<p>Given <code>n</code> versions, find the first bad one which causes all following versions to be bad.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> n = 5, bad = 4
<strong>Output:</strong> 4</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> n = 1, bad = 1
<strong>Output:</strong> 1</pre>
