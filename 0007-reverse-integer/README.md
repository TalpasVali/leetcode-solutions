<h2><a href="https://leetcode.com/problems/reverse-integer/">7. Reverse Integer</a></h2><h3>Medium</h3><hr>
<p>Given a signed 32-bit integer <code>x</code>, return <code>x</code> with its digits reversed. If reversing causes overflow, return <code>0</code>.</p>
<p><strong>Example 1:</strong></p>
<pre><strong>Input:</strong> x = 123
<strong>Output:</strong> 321</pre>
<p><strong>Example 2:</strong></p>
<pre><strong>Input:</strong> x = -123
<strong>Output:</strong> -321</pre>
